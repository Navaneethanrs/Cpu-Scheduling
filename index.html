First Come First Serve

The simplest scheduling algorithm. Processes are dispatched according to their arrival time on the ready queue.

Shortest Job First

Selects the waiting process with the smallest execution time to execute next.

Priority Scheduling

Each process is assigned a priority. Process with highest priority is to be executed first.

Round Robin

Each process is given a fixed time to execute, called a quantum.
